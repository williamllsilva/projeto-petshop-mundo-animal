# projeto-petshop-mundo-animal
🐾 PetShop Mundo Animal

Projeto desenvolvido durante meus estudos de HTML.

📖 Sobre o projeto

Este site simula um sistema simples de um petshop, contendo informações sobre serviços, produtos e um formulário de agendamento.

O objetivo principal foi praticar a estruturação de páginas HTML utilizando tags semânticas.

🛠️ Tecnologias utilizadas
HTML5
📚 O que foi praticado
Estrutura básica de páginas HTML
Uso de tags semânticas (header, main, section, footer)
Criação de menus de navegação com nav, ul, li e a
Inserção de imagens com img
Tabelas com table, tr, th e td
Formulários com form, input, label, select, option, textarea e button
Uso de radio e checkbox
Organização de conteúdo em seções
🚀 Objetivo de aprendizado

Este projeto faz parte da minha evolução como desenvolvedor front-end, focando inicialmente em HTML antes de avançar para CSS e JavaScript.
